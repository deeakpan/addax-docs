# Addax 101

Addax is a LitVM-based trading and liquidity interface built around concentrated liquidity pools and intent-based execution.

## Core concepts

- **Pools** hold token pairs and allow swaps.
- **Swaps** are executed through routing logic that can compare paths/fees.
- **ARIS intents** support offchain-signed, onchain-settled advanced orders.
- **LP positions** are represented as NFTs and can be managed by range.
